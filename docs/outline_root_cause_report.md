## 根因分析

### 现象
在高复杂度图形与较大描边粗细下，虚线/实线描边出现“斜穿直线的抄近路”问题，常规描边无此问题。

### 根因
虚线/实线描边使用的轮廓路径来自 Marching Squares 分段线段集合的无序拼接。原逻辑把线段视为无向图，在分支节点处随机选邻边，导致路径在复杂拓扑处跨越角点，形成错误的轮廓跳跃。描边越粗，膨胀掩膜越复杂，分支节点越多，错误概率更高。

### 修复策略
将线段组装替换为有方向约束的轮廓追踪。对每个节点按转角最小的规则选取下一条边，形成稳定、连续、无自交的外轮廓。最终仅保留面积最大的外环。

### 影响范围
仅影响虚线/实线描边的轮廓路径构建，常规描边仍使用相同掩膜但不受路径拼接错误影响。

## 修复效果
修复后虚线/实线描边在复杂图形与大线宽下不再出现“抄近路”，轮廓闭合正确，虚线段首尾对齐稳定。

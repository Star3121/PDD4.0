# Checklist

- [x] Direct Image Upload button allows uploading and adding images to canvas.
- [x] Square Frame can be added and masks images correctly.
- [x] Layer Panel displays all objects and reorders them correctly via drag-and-drop.
- [x] Text can be added and edited (font, size, color).
- [x] Wave Effect transforms text into a sinusoidal wave pattern.
- [x] Image filters (brightness, contrast, etc.) work and update in real-time.
- [x] Image strokes (regular, padded, dashed) render correctly.
